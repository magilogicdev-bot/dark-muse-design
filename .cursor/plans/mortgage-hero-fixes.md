### 2. Hero-секция: несоответствие типографики

#### Заголовок "Оформи ипотеку с Pobedonoscev GROUP"
- **Текущее**: `font-medium`, адаптивные размеры
- **Должно быть** (из Figma):
  - `fontSize: 128px` (на desktop)
  - `fontWeight: 500` (medium)
  - `lineHeight: 1.0234375em`
  - `letterSpacing: -1.5%`
  - `textCase: UPPER` (весь текст заглавными)
  - Размеры: 1787x335px на desktop

#### Описание (параграф)
- **Текущее**: `text-white/60`, адаптивные размеры
- **Должно быть**:
  - `fontSize: 29px`
  - `fontWeight: 400`
  - `lineHeight: 1.3103448275862069em`
  - `letterSpacing: -2.07%`
  - `opacity: 0.6`
  - Размеры: 980x157px на desktop

#### Кнопка "Хочу оформить ипотеку"
- **Текущее**: адаптивные размеры, `font-normal`
- **Должно быть**:
  - `fontSize: 36px`
  - `fontWeight: 400`
  - `lineHeight: 0.6666666666666666em`
  - `letterSpacing: -0.89%`
  - `borderRadius: 56px` (rounded-full)
  - Размеры: 568x120px на desktop
  - Текст внутри: 396x24px

#### Текст внизу "Мы работаем только с лидерами..."
- **Текущее**: адаптивные размеры
- **Должно быть**:
  - `fontSize: 32px`
  - `fontWeight: 400`
  - `lineHeight: 1.125em`
  - `letterSpacing: -3%`
  - Размеры: 944x63px на desktop

### 3. Проблемы с позиционированием изображения
- Изображение руки с домиком должно быть правильно масштабировано
- Логотипы должны быть точно позиционированы относительно изображения

### 4. Пропорции и отступы
- Нужно проверить gap между элементами
- Проверить padding секции
