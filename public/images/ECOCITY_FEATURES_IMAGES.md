# Ecocity Features Section Images

## Описание
Изображения для секции "ОСОБЕННОСТИ ПРОЕКТА" жилого комплекса Экогород 3.

## Необходимые изображения

1. **ecocity-features-playgrounds.png** - Детские и спортивные площадки
   - Node ID: `1080:7615`
   - Описание: Детская площадка с желтым и серым оборудованием

2. **ecocity-features-nature.png** - Свежий воздух и природа
   - Node ID: `1080:7628`
   - Описание: Зеленая растительность с деревьями и зданием на фоне

3. **ecocity-features-car-free.png** - Двор без машин
   - Node ID: `1080:7631`
   - Описание: Двор с фитнес-оборудованием и зелеными зонами

4. **ecocity-features-safety.png** - Безопасность
   - Node ID: `1080:7687`
   - Описание: Круглая скамейка на игровой поверхности

## Скачивание через универсальный скрипт

```bash
node scripts/download-figma-images.cjs \
  --file PoqoKMYQJFXDIWo8qbsKsj \
  --nodes 1080-7615,1080-7628,1080-7631,1080-7687 \
  --output public/images \
  --names ecocity-features-playgrounds.png,ecocity-features-nature.png,ecocity-features-car-free.png,ecocity-features-safety.png
```
