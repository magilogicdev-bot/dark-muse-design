# Изображения планировок квартир

## Инструкция по экспорту из Figma

1. Откройте файл Figma: https://www.figma.com/design/dbhRRPEagcwqR97v2vLgAd/ALFA-12.15?node-id=1034-460&m=dev
2. Найдите карточки планировок квартир
3. Экспортируйте каждую планировку:
   - Выберите изображение планировки
   - В правой панели найдите "Export"
   - Выберите формат PNG или SVG
   - Нажмите "Export"
   - Сохраните файлы в эту папку с именами:
     - `plan-1.png` (первая планировка)
     - `plan-2.png` (вторая планировка)
     - `plan-3.png` (третья планировка)
     - `plan-4.png` (четвертая планировка)
